# 🏦 Microservices Banking System

A scalable digital banking application built using Spring Boot Microservices, Flutter, JWT Authentication, MySQL, Eureka Service Discovery, and API Gateway.

## 🚀 Features

### Authentication

* User Registration
* User Login
* JWT Authentication
* Secure API Access

### Banking Operations

* Automatic Account Creation
* Balance Inquiry
* Deposit Money
* Withdraw Money
* Transfer Money
* Transaction History

### Architecture

* Microservices-Based Design
* Eureka Service Discovery
* API Gateway
* MySQL Database

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT
* Spring Data JPA
* Eureka Server
* API Gateway
* MySQL

### Frontend

* Flutter
* Dart
* Shared Preferences

## 🏗️ Architecture

```text
Flutter App
     │
     ▼
API Gateway
     │
 ┌───┼─────────────┐
 ▼   ▼             ▼

User Service
Account Service
Transaction Service

     │
     ▼

MySQL Database

     ▲
     │

Eureka Server
```
